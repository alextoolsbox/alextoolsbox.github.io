# AlexToolsBox

AlexToolsBox è un sito di guide pratiche dedicate a fintech, app digitali e strumenti online utili per risparmiare o ottenere vantaggi dalle piattaforme digitali.

Il progetto nasce con l'obiettivo di spiegare in modo semplice e concreto come utilizzare servizi come Revolut e Microsoft Rewards.
